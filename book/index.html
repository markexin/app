  • 读书总结
    • 前端架构设计
      • 前端架构的核心
        • 前端架构是一系列工具和流程的集合，旨在提升前端代码的质量，并实现高效、可持续的工作流。
      • 前端架构的内容
        • 体系设计
        • 工作规划
          • 版本控制器
          • 任务调度器
          • CSS处理器
          • 文档工具
          • 测试组件
          • 服务器自动化工具
        • 监督跟进
          • 持续优化工作流程
      • 前端架构原则
        • 代码
          • HTML
            • 程序式标记(自动化:100%,可控程度:0%)
              • div一把梭，层级嵌套较深。典型的ant design搭建CMS管理系统
            • 静态标记(自动化:0%,可控程度:100%)
              • 不使用类名，利用html属性一把梭
            • 模块化标记(自动化:100%,可控程度:100%)
              • 以模块为单位，利用class智能语义化
          • CSS
            • OOCSS
              • 分离结构和外观
              • 分离容器和内容
            • SMACSS
              • 携带is状态类名的模块
            • BEM
              • 是一种CSS命名规范，块名+元素+修饰符
          • javascript
          • Red Hat代码
        • 流程
        • 测试
        • 文档
