      • 类型、值和变量
        • 原始类型
          • 数字
            • （整数）边界：-2^53 ~ 2^53, 超出边界，无法保证精度。
            • （按照位运算）边界：32位整数。
            • 当按位运算溢出之后，会用 Infinity 表示无穷大。
            • toString() –> 转换成字符串
            • valueOf() –> 获取值
          • 字符串
            • 一个字符就是一个16位的值。
            • 字符串可以通过索引来访问，所以可以当做一个只读数组。
            • toString() –> 转换成字符串
            • valueOf() –> 获取值
          • 布尔值
            • 任意的javascript的值都可以转换为布尔值。
            • 字符串可以通过索引来访问，所以可以当做一个只读数组。
            • toString() –> 转换成字符串
            • valueOf() –> 获取值
          • null & undefined
            • null: 表示没有这个值。
            • undefined: 表示值未定义。
          • BigInt
          • Symbol

        • 体系设计

          • CSS处理器
          • 文档工具
          • 测试组件
          • 服务器自动化工具
        • 监督跟进
          • 持续优化工作流程
      • 前端架构原则
        • 代码
          • HTML
            • 程序式标记(自动化:100%,可控程度:0%)
              • div一把梭，层级嵌套较深。典型的ant design搭建CMS管理系统

              • 不使用类名，利用html属性一把梭
